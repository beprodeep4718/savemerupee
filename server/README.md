# SaveMoney — Backend

This folder contains a simple Node.js + TypeScript Express backend for local development.

Quick start

1. Install dependencies:

```powershell
cd "server"
npm install
```

2. Run in development mode (auto restarts):

```powershell
npm run dev
```

3. Build and run production:

```powershell
npm run build
npm start
```

API routes

- `GET /` — basic health check
- `GET /api/health` — API health
- `GET /api/hello` — example endpoint
